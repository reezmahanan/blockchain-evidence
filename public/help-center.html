Getting Started with EVID-DGC

What is EVID-DGC?

EVID-DGC is a blockchain-based evidence management system designed for law enforcement, legal professionals, and court officials. It ensures evidence integrity through immutable records and provides role-based access control.

Key Features:

  • Secure Evidence Storage: All evidence is cryptographically secured
  • Chain of Custody: Complete audit trail of evidence handling
  • Role-Based Access: Different permissions for different user types
  • Blockchain Verification: Tamper-proof evidence integrity

Cases progress through different statuses based on investigation progress:

Open

Case is actively being investigated

In Progress

Case investigation is complete

4

5

Verify Upload

System will generate a cryptographic hash for integrity verification

Evidence Verification

All evidence is automatically verified using blockchain technology:

  • Hash Generation: Each file gets a unique cryptographic fingerprint
  • Blockchain Storage: Hash is stored on immutable blockchain
  • Integrity Check: Files are verified against blockchain records
  • Tamper Detection: Any changes to files are immediately detected

Chain of Custody

The system automatically maintains a complete chain of custody record:

  • Who uploaded the evidence
  • When it was uploaded
  • Who has accessed the evidence
  • Any modifications or annotations
  • Download history with watermarks

Investigator

  • Upload evidence
  • Generate reports
  • Manage investigations

Forensic Analyst

Analyze evidence and create technical reports

  • Analyze evidence
  • Generate forensic reports

Legal Professional

  • Review case documentation
  • Access legal evidence
  • Generate legal reports
  • Update case status

Security & Privacy

Data Security

EVID-DGC implements multiple layers of security:

  • Encryption: All data is encrypted in transit and at rest
  • Blockchain: Evidence hashes stored on immutable blockchain
  • Access Control: Role-based permissions system
  • Audit Logs: Complete activity tracking
  • Watermarking: Downloaded evidence is watermarked

Password Security

Best Practices:

  • Use at least 8 characters
  • Include uppercase and lowercase letters
  • Add numbers and special characters
  • Don't reuse passwords from other accounts
  • Change passwords regularly

Troubleshooting

Common Issues

I forgot my password. How do I reset it?

Click "Forgot Password?" on the login page, enter your email address, and follow the instructions in the reset email.

Why can't I upload evidence?

Check that: 1) You have the correct role permissions, 2) File size is under 100MB, 3) File type is supported, 4) You're connected to the internet.

How do I change my role?

Only administrators can change user roles. Contact your system administrator to request a role change.

What browsers are supported?

EVID-DGC works best with modern browsers: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+. For MetaMask features, use Chrome or Firefox.
