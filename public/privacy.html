Privacy Policy

Last updated: February 2026

1. Information We Collect

Blockchain Data

  • Wallet Addresses: Your MetaMask wallet address for authentication
  • Evidence Hashes: Cryptographic hashes of evidence files (NOT the actual files)
  • Transaction Records: Blockchain transaction hashes for audit trails
  • Timestamps: When evidence was submitted or accessed

Application Data

  • User Profile: Name, role, department, jurisdiction
  • Case Information: Case numbers, titles, descriptions
  • Activity Logs: System access and action records
  • Evidence Metadata: File names, types, sizes (stored securely off-chain)

2. What Goes On-Chain vs Off-Chain

On Blockchain (Public)

  • Evidence file hashes (SHA-256)
  • Wallet addresses
  • Timestamps
  • Transaction hashes

Off-Chain (Private)

  • Actual evidence files
  • Personal information
  • Case details
  • User profiles

3. Data Security

  • Encryption: All data encrypted in transit and at rest
  • Access Controls: Role-based permissions and multi-factor authentication
  • Blockchain Security: Immutable records on distributed ledger
  • Regular Audits: Security assessments and penetration testing

4. Your Rights

  • Access: Request information about your data
  • Correction: Update inaccurate personal information
  • Deletion: Request deletion where legally permissible
  • Portability: Export your data in standard formats

Note: Some rights may be limited due to legal evidence preservation requirements.
