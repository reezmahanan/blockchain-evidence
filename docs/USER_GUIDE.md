- Browse available cases
- View case summaries
- No evidence upload/download

**Access**: Create cases, upload evidence

- Upload evidence files (PDF, images, videos, audio, documents)
- Assign team members
- Track case progress

### 3. Forensic Analyst

**Access**: Analyze evidence, generate reports

- Analyze uploaded evidence

- Verify evidence integrity

### 4. Legal Professional

- Verify chain of custody

### 5. Court Official

- Schedule hearings
- Update case status

- Access case timeline

### 6. Evidence Manager

### 7. Auditor

### 8. Administrator

## Common Tasks

### Login Process

1. Go to system homepage
2. Choose login method:
   - **MetaMask**: Click "Connect Wallet", approve connection
   - **Email**: Click email login, enter credentials
3. Complete registration if first time
4. Access role-specific dashboard

### Evidence Upload (Investigators/Analysts)

1. Navigate to evidence upload
2. Select case or create new case
3. Choose file (max 100MB)
4. Add description and metadata
5. Submit upload
6. Verify upload success

### Case Management

1. **Create Case**: Fill case details, assign priority
2. **Update Status**: Change case status as it progresses
3. **Assign Team**: Add team members to case
4. **Track Progress**: Monitor case timeline

### Download Evidence

1. Navigate to case evidence
2. Select evidence item
3. Click download (watermarked version)
4. Verify download integrity

## Troubleshooting

### Login Issues

- **MetaMask not connecting**: Install MetaMask extension
- **Wrong network**: Switch to Polygon network
- **Email login fails**: Check credentials, reset password

### Upload Issues

- **File too large**: Max 100MB per file
- **Unsupported format**: Use PDF, images, videos, audio, documents
- **Upload fails**: Check network connection, try again

### Access Issues

- **Permission denied**: Contact admin for role verification
- **Page not loading**: Clear browser cache, refresh

### System Errors

- **500 errors**: Contact system administrator
- **Network errors**: Check internet connection
- **Database errors**: System may be under maintenance

## Security Best Practices

- Never share wallet private keys
- Use strong passwords for email accounts
- Log out when finished
- Report suspicious activity
- Keep MetaMask updated
